# Exam Fee Comparison

This section provides a basic comparison of the approximate registration fees for major international standardized exams required for studying abroad.

| Exam  | Approximate Fee in India |
| ----- | ------------------------ |
| SAT   | ₹9,000 – ₹11,000         |
| GRE   | Around ₹22,000           |
| GMAT  | Around ₹23,000           |
| TOEFL | Around ₹16,000           |
| IELTS | Around ₹17,000           |

These costs can increase further if students need to retake exams or send additional score reports to universities.
